---

## 1. Descripción del Proyecto

SIRMA_JG es una aplicación de escritorio desarrollada en Java con Swing y Graphics2D que permite llevar un registro detallado y predictivo del mantenimiento de vehículos. El sistema gestiona propietarios, vehículos y su historial de servicios, ofreciendo una interfaz gráfica moderna e intuitiva.

---

## 2. Requisitos de Instalación

*   **IDE:** IntelliJ IDEA (o cualquier otro IDE compatible con Java).
*   **JDK:** Java Development Kit (JDK) versión 17 o superior.

---

## 3. Configuración y Ejecución

1.  Clonar este repositorio de GitHub o descargar el archivo ZIP.
2.  Abrir el proyecto desde IntelliJ IDEA mediante la opción `File > Open...` y seleccionando la carpeta del proyecto.
3.  Una vez el proyecto esté abierto y cargado, buscar el archivo `Main.java`.
4.  Hacer clic derecho en el archivo `Main.java` y seleccionar la opción `Run 'Main.main()'`.
